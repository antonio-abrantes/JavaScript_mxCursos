Curso de JavaScript - Boas praticas

* Boas praticas
* Evitar uso de variaveis globais
* Preferencia por declarar as variavis no inicio do codigo
* Sempre inicializar as variaveis
* Só delcarar variaveis do tipo objeto se realmente for necessario
* Dar preferencia para o operador de comparação '===' e '!=='
* Sempre utilizar uma opção default no Swhith-Case
